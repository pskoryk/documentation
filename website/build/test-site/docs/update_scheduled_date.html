Update date in scheduled campaign

PATCH Update date in a scheduled campaign

Definition:
Method: GET
Endpoint:https://uapi.gravitec.net/api/v3/push/[campaignID]?newSendDate=[date]
Request parameters:

Parameter

Description

Requirements

Limitations

campaignID

Campaign ID

Required

Push campaign Id returned in method "Send Push Notification V3"

date

New scheduled date

Required

Time must be inUNIX_TIMEformat, which is essentially
the number of seconds that have elapsed since 00:00:00 Coordinated Universal Time (UTC), Thursday, 1 January 1970, minus the number of leap seconds that have taken place since then

NOTE: All request parameters names and values are case-sensitive.
Request URL example:
https://uapi.gravitec.net/api/v3/push/1603861504654835713?newSendDate=1509058800
